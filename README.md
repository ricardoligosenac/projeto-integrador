## 🚀 Como Usar

1. **Acesso ao Sistema**
   - Navegue até `http://localhost/projeto-integrador`
   - Faça login com suas credenciais

2. **Gerenciar Clientes**
   - Após login, acesse o módulo de clientes
   - Utilize as opções de criar, visualizar, editar ou deletar clientes
